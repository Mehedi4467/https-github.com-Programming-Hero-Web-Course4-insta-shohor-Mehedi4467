1. How does javascript Work?

Ans: All modern browsers today support JavaScript and have a JavaScript engine. Which is the V8 engine in the case of Chrome. Whenever we run a JavaScript program inside a web browser, the browser engine receives the JavaScript code. . Then go through the code parser. Parser is the first step of the engine. Parser examines JavaScript code for syntactic errors in a line-by-line manner. Then go through AST (it stands for Abstract Syntax Tree). It then converts the code into machine code. The converted code is sent to the system for execution. Eventually this machine runs the code on the system.

2. Difference between local storage and session storage?

Ans: Local storage and shortage storage are worked for saving any object or data as a key value pairs in browser but they have some basic different. Like-

  • Local storage save data until it won't be deleted by javascript code or browser. On the otherhand session storage save data only when the tab is open.if the tab is closed then the data will be deleted.
  • Local storage data can be accessed to any tab in browser but session storage data need same tab for getting access.
  • The capacity of local Storage is 10 mb on the contrary session storage capacity is 5 mb.
